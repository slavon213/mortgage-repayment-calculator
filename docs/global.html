calculate(amount, term, rate, mortgageTypeopt) → {Array.<string>}

Розраховує щомісячний платіж і загальну суму виплат за іпотекою. Підтримує два типи іпотеки: - "repayment": класичний ануїтетний платіж з поступовим погашенням основної суми. - "interest": лише виплата відсотків без погашення основної суми.
Parameters:
Name Type Attributes Default Description
amount number Сума кредиту (основна сума).
term number Термін кредиту в роках.
rate number Річна процентна ставка (у відсотках).
mortgageType string <optional>
 "repayment" Тип іпотеки: "repayment" або "interest".
Source:
  • script.js, line 365
Returns:
- Масив з двох значень: [0] — щомісячний платіж (округлений до 2 знаків), [1] — загальна сума виплат за весь термін (округлена до 2 знаків).
Type
Array.<string>
Example
calculate(100000, 25, 5); // ["584.59", "175376.52"]
calculate(100000, 25, 5, "interest"); // ["416.67", "125000.00"]
